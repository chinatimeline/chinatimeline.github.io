# 中国大事件时间线
时间线可以帮助我们清晰地勾勒一个纷繁复杂的历史过程，帮助人们理解过去、预测未来。最重要的是帮助整个社会及时有效的影响当下。
希望通过这个开源项目还原各种重大历史进程，例如：
- 进行中的[文革2.0](#ideologicalwar)
- 进行中的[中美贸易战](#tradewar)
- 文化大革命
- 六四天安门事件
- 中国的改革开放、加入WTO、一带一路等等。
